<ul>
  <li>One strip </li>
  <li>Two strips </li>
</ul><hr>
<h3>Here is what I used:</h3>
<ul>
  <li>Arduino Uno</li>
  <li>Small breadboard</li>
  <li>Random addressable GRB strip from china</li>
  <li>1 resistor (not needed, but it's better to have one)</li>
  <li>Jumper wires</li>  
</ul>
